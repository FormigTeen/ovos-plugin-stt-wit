Ovos Wit STT  Plugin

## Description

OpenVoiceOS companion plugin for [OpenVoiceOS STT Server](https://github.com/OpenVoiceOS/ovos-stt-http-server)

## Install

```bash
pip install git+https://github.com/FormigTeen/ovos-stt-plugin-wit
```

## Configuration

```json
  "stt": {
    "module": "ovos-stt-plugin-wit",
    "ovos-stt-plugin-wit": {
        "key": "YOUR-KEY-HERE"
    }
 }
```
